# Tania Ramzi Ajab Portfolio

Static GitHub Pages portfolio for Tania Ramzi Ajab, focused on medical AI, vision-language models, computer vision, trustworthy AI evaluation, and applied engineering projects.

## Files

- `index.html` - complete portfolio content and SEO metadata
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and footer year
- `assets/profile.jpeg` - portrait image
- `assets/Tania_Ajab_CV.pdf` - downloadable CV
- `robots.txt` and `sitemap.xml` - search indexing helpers

## Deploy on GitHub Pages

1. Create a GitHub repository named `portfolio` under the `Tania-Ajab` account.
2. Upload all files from this folder.
3. In the repository, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/root`, then save.
6. The expected site URL is `https://tania-ajab.github.io/portfolio/`.

## Google Indexing

1. After GitHub Pages is live, open Google Search Console.
2. Add the live GitHub Pages URL as a URL-prefix property.
3. Submit `https://tania-ajab.github.io/portfolio/sitemap.xml`.
4. Use the URL inspection tool to request indexing for the homepage.

If the repository name or custom domain changes, update the URL inside `sitemap.xml` and the structured-data `url` in `index.html`.
